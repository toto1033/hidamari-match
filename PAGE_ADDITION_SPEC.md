## Step 1: 施設IDをUUID形式に変更

`src/data/facilities.ts` の全施設の `id` フィールドを
連番（"1"・"2"）からUUID形式に変更する。

```typescript
// 変更前
id: "1"

// 変更後（このような形式に）
id: "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
```

uuidパッケージを使わず、以下の形式で固定値として記述する：
各施設に固有のUUIDを割り当てること。
